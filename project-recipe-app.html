UI/UX Design

Recipe App

Designed a recipe recommendation app for iOS with a strong focus on user experience and design planning.

Overview

This project focused on the complete design process of an iOS recipe application. From conducting desk research to creating high-fidelity prototypes, the goal was to create an intuitive and visually appealing interface that helps users discover new meals.

Key Features

User Research

In-depth analysis of user needs and pain points in cooking.

Wireframing

Structural design of the app's layout and flow.

Prototyping

Interactive Figma prototypes demonstrating user interactions.

iOS HIG Compliance

Adherence to Apple's Human Interface Guidelines for a native feel.

Tech Stack

Figma UI/UX User Research Prototyping iOS HIG
